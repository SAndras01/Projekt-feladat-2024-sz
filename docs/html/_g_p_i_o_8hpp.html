Detailed Description

A simple GPIO control interface for STM32.

This file provides a class interface for controlling GPIO pins on STM32 devices using HAL libraries. It allows setting and reading the state of GPIO pins.

The GPIO class abstracts basic GPIO operations such as digital write and read.

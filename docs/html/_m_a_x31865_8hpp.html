Detailed Description

Interface for MAX31865 RTD-to-Digital converter.

This file provides functions and macros to interact with the MAX31865 sensor for temperature measurement.

The MAX31865 is a resistance-to-digital converter optimized for PT100 and PT1000. It can operate in both 2-wire, 3-wire, and 4-wire configurations, with all of which are supported.
